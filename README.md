# 📰 Newsletter Maker

高齢者福祉施設向け新聞自動作成ツール

## 特徴

- 🖼️ 写真を選ぶだけで自動レイアウト
- 🎨 12ヶ月分のカラフルなテンプレート
- ✏️ 文章自動生成&手動編集可能
- 📄 A4カラー印刷対応
- 💻 完全オフライン動作（API不要）

## 技術スタック

- Electron (デスクトップアプリ)
- HTML/CSS/JavaScript
- jsPDF (PDF生成)
